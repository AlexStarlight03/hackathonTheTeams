Pour lancer le projet:

dans un terminal entrer les commandes suivantes pour avoir les installations nécessaires:
cd .\Backend\
npm install

puis entrer cette commande pour lancer le backend:
npm run dev

ensuite dans un autre terminal, entrer les commandes suivantes:
cd ..\Frontend\
npm install

puis entrer cette commande pour lancer le frontend:
npm run dev

sur votre navigateur web de votre choix, rendez-vous à http://localhost:5173/ pour profiter de notre projet!
